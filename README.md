
# Timbo New

Crie seus projetos facilmente através de uma CLI
## Instalação

Instale timbo-new com:

### npm
```bash
npm install timbo-new -g
```
    
## Próximos passos

Verifique a versão do seu projeto com:
```bash
new -v

// 0.1.0-alpha
```
